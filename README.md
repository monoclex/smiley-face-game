# Contributing

_NOTE: these instructions are the bare minimum needed. At a future point, these instructions will be more comprehensive._

Make sure you have the following setup:

- pnpm
- docker
- docker compose

Then, run the following to start the project:

```shell
pnpm i

docker-compose up -d

cd packages/client
pnpm serve
```
